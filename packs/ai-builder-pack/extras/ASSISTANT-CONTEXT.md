# Assistant Context

Use this as a starter instruction for AI-assisted work inside a seeded project.

## Project Context To Read First

- `foundation/PROJECT-RULES.md`
- `foundation/CODE-STANDARDS.md`
- `foundation/DEFINITION-OF-DONE.md`
- `memory/PROJECT-CONTEXT.template.md`
- `memory/CURRENT-STATE.template.md`
- `memory/DECISION-LOG.template.md`
- `skills/ai-collaboration-skill.md`

## Working Instructions

- Inspect existing patterns before editing.
- Keep changes scoped to the requested task.
- Explain uncertainty and tradeoffs.
- Add tests or verification notes.
- Update docs or memory when decisions, setup, behavior, or current state change.
- Use the code review skill before finalizing.

## Session Closeout

At the end of a meaningful session, update:

- current state,
- decision log,
- known issues,
- session notes,
- next best actions.
