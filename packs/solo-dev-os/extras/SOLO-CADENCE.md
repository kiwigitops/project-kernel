# Solo Cadence

Use this lightweight cadence to keep a solo project from losing context.

## Start Of Session

- Read `memory/CURRENT-STATE.template.md`.
- Read `memory/OPEN-QUESTIONS.template.md`.
- Choose one priority from `memory/PRIORITIES.template.md`.

## During Session

- Keep scope small.
- Record decisions as they happen.
- Add known issues instead of chasing every side quest.

## End Of Session

- Update current state.
- Add session notes.
- Write the next best action.
- Move deferred work into roadmap or priorities.
