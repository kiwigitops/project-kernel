# Solo Dev OS Pack

Use this pack for a long-running solo project where continuity matters.

## Best For

- Personal apps.
- Indie products.
- Long experiments.
- Side projects that pause and resume.

## Included Modules

- `foundation`
- `docs`
- `memory`
- `workflow`
- `philosophy`

## What It Gives You

- Strong project defaults.
- Documentation and roadmap structure.
- Memory files for goals, state, assumptions, decisions, known issues, and session notes.
- Repeatable planning, implementation, review, and maintenance loops.
- Philosophy that keeps the project clear and maintainable.

## First Five Minutes

1. Fill project goals in `memory/PROJECT-CONTEXT.template.md`.
2. Fill `memory/CURRENT-STATE.template.md`.
3. Fill setup commands.
4. Create the first roadmap slice.
5. End the first work session by updating session notes.

## Install

```bash
node scripts/install-pack.js packs/solo-dev-os ../target-project
```
