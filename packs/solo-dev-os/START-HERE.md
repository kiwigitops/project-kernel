# Start Here: Solo Dev OS

This pack helps a solo developer keep momentum across time.

## First Actions

- Fill `memory/PROJECT-CONTEXT.template.md`.
- Fill `memory/CURRENT-STATE.template.md`.
- Fill `memory/PRIORITIES.template.md`.
- Read `workflow/planning-workflow.md` before the first non-trivial feature.
- Read `philosophy/continuity-principles.md` before deciding what context to preserve.

## Session Habit

At the end of each meaningful work session:

- update current state,
- note decisions,
- capture known issues,
- write next best actions.
