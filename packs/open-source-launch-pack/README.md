# Open-Source Launch Pack

Use this pack when a repo is preparing for public use, contributors, releases, and reuse.

## Best For

- Public libraries.
- Developer tools.
- Documentation projects.
- Community-maintained starters.

## Included Modules

- `foundation`
- `docs`
- `workflow`
- selected launch extras.

## What It Gives You

- README, contributing, changelog, roadmap, setup, architecture, decisions, and FAQ templates.
- Project rules and quality bar.
- Issue triage and release workflows.
- Launch checklist for public readiness.

## First Five Minutes

1. Turn `docs/README.template.md` into a public README.
2. Fill `docs/CONTRIBUTING.template.md`.
3. Start `docs/CHANGELOG.template.md`.
4. Fill `docs/ROADMAP.template.md`.
5. Complete `extras/LAUNCH-CHECKLIST.md`.

## Install

```bash
node scripts/install-pack.js packs/open-source-launch-pack ../target-project
```
