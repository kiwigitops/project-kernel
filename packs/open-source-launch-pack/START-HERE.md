# Start Here: Open-Source Launch

This pack prepares a project for public readers and contributors.

## First Actions

- Make the README clear above the fold.
- Add setup commands.
- Add contribution expectations.
- Add changelog structure.
- Add roadmap direction.
- Run the launch checklist.

## Launch Standard

A public repo is ready when a stranger can understand what it is, install it, evaluate whether it fits their use case, contribute safely, and see where the project is going.
