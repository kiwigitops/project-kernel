# Start Here: Minimal Starter

Copy or install this pack when the project needs structure but not a full operating system yet.

## First Actions

- Read `foundation/START-HERE.md`.
- Fill project identity in `foundation/PROJECT-RULES.md`.
- Fill commands in `foundation/SETUP-CHECKLIST.md`.
- Keep `foundation/DEFINITION-OF-DONE.md` visible during the first implementation pass.

## Upgrade Path

When the project becomes durable, add:

- `modules/docs`
- `modules/memory`
- `modules/workflow`
