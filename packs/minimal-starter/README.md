# Minimal Starter Pack

Use this pack when you want the smallest serious foundation for a new project.

## Best For

- Prototypes that may become real.
- Small internal tools.
- Experiments that still need setup and quality rules.

## Included Modules

- `foundation`

## What It Gives You

- Project rules.
- Code standards.
- File and naming conventions.
- Setup checklist.
- Definition of done.

## First Five Minutes

1. Fill `foundation/PROJECT-RULES.md`.
2. Fill required commands in `foundation/SETUP-CHECKLIST.md`.
3. Pick naming and folder conventions.
4. Define what "done" means for the first version.

## Install

```bash
node scripts/install-pack.js packs/minimal-starter ../target-project
```
