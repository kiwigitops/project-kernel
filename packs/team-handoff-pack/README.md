# Team Handoff Pack

Use this pack when a project needs to be understood by someone who did not build it.

## Best For

- Team onboarding.
- Maintenance transfers.
- Client handoff.
- Internal repos with unclear ownership.

## Included Modules

- `docs`
- `memory`
- `workflow`
- selected handoff extras.

## What It Gives You

- Architecture, setup, contribution, and FAQ templates.
- Current state, constraints, open questions, decisions, and known issues.
- Review, maintenance, and issue triage workflows.
- A handoff checklist.

## First Five Minutes

1. Fill architecture summary.
2. Fill current state.
3. List blockers and open questions.
4. Capture known issues and workarounds.
5. Fill the handoff checklist in `extras/HANDOFF-CHECKLIST.md`.

## Install

```bash
node scripts/install-pack.js packs/team-handoff-pack ../target-project
```
