# Handoff Checklist

Use this checklist before transferring ownership or onboarding a new contributor.

- [ ] Setup instructions are current.
- [ ] Architecture summary reflects reality.
- [ ] Current state is updated.
- [ ] Known issues include workarounds.
- [ ] Open questions have owners or next actions.
- [ ] Important decisions are linked.
- [ ] Common commands are documented.
- [ ] Release or deployment flow is explained.
- [ ] The next best actions are explicit.
