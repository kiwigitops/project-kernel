# Start Here: Team Handoff

This pack is for making the next maintainer productive quickly.

## First Actions

- Fill `docs/SETUP.template.md`.
- Fill `docs/ARCHITECTURE.template.md`.
- Fill `memory/CURRENT-STATE.template.md`.
- Fill `memory/OPEN-QUESTIONS.template.md`.
- Fill `memory/KNOWN-ISSUES.template.md`.
- Complete `extras/HANDOFF-CHECKLIST.md`.

## Handoff Standard

A handoff is ready when a new contributor can run the project, understand the architecture, see what is in progress, and know the next useful action.
