# Start Here: Documented Starter

Use this pack when documentation quality matters immediately.

## First Actions

- Read `foundation/START-HERE.md`.
- Convert `docs/README.template.md` into a project README.
- Fill `docs/SETUP.template.md` with commands a new contributor can run.
- Add architecture notes before the first major refactor.
- Use `workflow/review-workflow.md` before merge or handoff.

## Keep Current

When setup, architecture, or public behavior changes, update docs in the same change.
