# Module Map

Use this map when deciding which modules to copy.

```text
foundation
  defines rules and quality
  feeds workflow and review

docs
  explains project to humans
  links architecture, setup, roadmap, contribution

memory
  preserves context and decisions
  feeds AI-assisted work, handoff, maintenance

skills
  provides reusable procedures
  supports debugging, testing, review, refactor, release

workflow
  defines project loops
  turns standards into repeated practice

philosophy
  explains why the system works this way
  keeps choices consistent over time
```

## Minimal Connections

- Foundation plus docs gives a project understandable structure.
- Foundation plus memory gives durable working context.
- Memory plus skills improves AI-assisted development.
- Workflow plus skills makes repeated work reviewable.
- Philosophy plus foundation keeps standards from becoming arbitrary.
