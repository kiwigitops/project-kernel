# Changelog

All notable changes to Project Kernel are tracked here.

## 0.2.0 - 2026-04-13

### Added

- Integrated sourced and curated Claude skills for design sprint facilitation, UI/UX system generation, iOS platform design, usability audits, retention hook analysis, and visual UI refactoring.
- Added sourced skills index and third-party notices.
- Added product design workflow that routes product, UX, UI, platform, and retention work to the right skill.
- Added design review memory template for preserving UX/UI findings and design system decisions.

## 0.1.0 - 2026-04-13

### Added

- Initial Project Kernel system.
- Foundation, docs, memory, skills, workflow, and philosophy modules.
- Curated packs for minimal starts, documentation-heavy starts, solo development, team handoff, open-source launch, and AI-assisted building.
- Example use cases for seeded projects.
- Lightweight Node scripts for pack installation, structure validation, and README index generation.
