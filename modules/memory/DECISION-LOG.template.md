# Decision Log

Record decisions while the reasoning is still fresh. Keep entries brief but useful.

## Entry Format

### [YYYY-MM-DD] - [Decision Title]

**Status:** proposed / accepted / superseded / retired

**Context:** [What situation required a decision?]

**Decision:** [What was chosen?]

**Reasoning:** [Why this option?]

**Alternatives considered:**

- [Alternative]
- [Alternative]

**Consequences:**

- [Benefit]
- [Tradeoff]

**Follow-up:** [Action or review date]
