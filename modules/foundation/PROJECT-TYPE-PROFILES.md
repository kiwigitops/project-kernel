# Project Type Profiles

Use these profiles to tune the foundation module for the kind of project you are starting.

## App

Emphasize:

- user-facing behavior,
- setup and environment variables,
- deployment or release flow,
- error handling,
- accessibility and usability,
- product roadmap.

Recommended modules:

- `foundation`
- `docs`
- `memory`
- `workflow`

## Library

Emphasize:

- public API stability,
- examples,
- versioning,
- changelog quality,
- compatibility policy,
- test coverage across supported environments.

Recommended modules:

- `foundation`
- `docs`
- `workflow`
- selected `skills/testing-skill.md`

## CLI

Emphasize:

- command usage,
- flags and examples,
- error messages,
- exit codes,
- installation path,
- release process.

Recommended modules:

- `foundation`
- `docs`
- `workflow`
- selected `skills/ship-readiness-skill.md`

## Docs-Only Project

Emphasize:

- information architecture,
- reader paths,
- contribution rules,
- changelog for major doc changes,
- maintenance cadence.

Recommended modules:

- `docs`
- `workflow`
- `philosophy`

## Experimental Repo

Emphasize:

- goals,
- constraints,
- assumptions,
- learning milestones,
- what is intentionally not production-ready.

Recommended modules:

- `foundation`
- `memory`
- selected `workflow/planning-workflow.md`
