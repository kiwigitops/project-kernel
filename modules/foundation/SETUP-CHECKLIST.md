# Setup Checklist

Use this checklist to make setup repeatable for humans and AI tools.

## Required Tools

- Runtime: `[Node/Python/Ruby/Go/etc.]`
- Package manager: `[npm/pnpm/yarn/pip/uv/etc.]`
- Database or services: `[name/version]`
- Environment manager: `[optional]`

## First-Time Setup

- [ ] Clone or create the repository.
- [ ] Install required runtime.
- [ ] Install dependencies with `[command]`.
- [ ] Copy environment variables from `[example file]`.
- [ ] Run database or external services if needed.
- [ ] Run setup command: `[command]`.
- [ ] Run tests: `[command]`.
- [ ] Run development server or main entry point: `[command]`.

## Setup Verification

Setup is complete when:

- dependencies install cleanly,
- tests run,
- the project starts locally,
- required environment variables are documented,
- and a new contributor can reproduce the steps without private context.
