# Skills Module

Skills are reusable operating procedures. Use them when a project needs a repeatable way to debug, review, refactor, document, test, plan, ship, or work with AI tools.

## Read First

1. [`feature-planning-skill.md`](feature-planning-skill.md)

3. [`code-review-skill.md`](code-review-skill.md)
4. [`refactor-skill.md`](refactor-skill.md)
5. [`ship-readiness-skill.md`](ship-readiness-skill.md)

## How To Use A Skill

1. Open the skill when the matching work begins.
2. Follow the process in order unless the project context justifies a change.
3. Produce the expected output.
4. Update related docs or memory files.
